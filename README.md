FlipClock
=========

A flip clock screensaver supported by SDL2.
-------------------------------------------

![flipclock.png](flipclock.png)

# Usage

## Linux/macOS:

1. Install sdl2, sdl2_ttf.
2. `mkdir build && cd build && cmake -DCMAKE_BUILD_TYPE=Debug .. && make`
3. `./bin/flipclock`
